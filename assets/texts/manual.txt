Rooms 
    • **By default match is positve** (ascending order)
    > `/match +1` __to search for a party with less users__
    > `/match -1` __to search for a party with more users__
    • **By default the party size is +2** (maximum is 10).
    > `/party -2` (negative) __to create a hidden party.__
    > `/party 10` (positive) __to create a public party.__
    • **To disconnect** just send /unmatch
